# 📱 移动宽带服务管理系统 (Mobile Broadband Service System)

## 📖 项目简介
本项目是一款基于 **Spring Boot + Vue** 的全栈宽带业务管理系统。采用 **MySQL** 存储数据，前端基于  **Vue + Element UI **组件库与 HTML/CSS/JS 开发。旨在实现宽带业务全流程的数字化管理。

## 🚀 核心功能模块

### 1. 客户模块 (Client)
* **实名登记 & 登录**：安全的用户身份验证。
* **智能套餐推荐**：基于用户历史月租数据，通过算法智能推荐最适合的宽带套餐。
* **自主选餐与结算**：透明化的套餐列表展示，支持在线选购与费用支付。
* **年度账单分析**：图形化展示用户消费数据，为下一年决策提供参考。
* **故障报修**：在线提交宽带故障申请。

### 2. 维修人员模块 (Maintenance)
* **任务看板**：实时查看分配的安装或报修申请地址。
* **工单处理**：上传故障原因诊断，更新任务完成状态。

### 3. 管理员模块 (Admin)
* **全生命周期管理**：审核用户信息，管理订单状态。
* **智能派单**：将安装/维修任务分配给指定人员。
* **财务对账**：管理客户缴费记录与费用核算。

## 🛠️ 技术栈
* **后端**：Java, Spring Boot
* **前端**：Vue.js, Layui, HTML/CSS/JavaScript
* **数据库**：MySQL
